# zl-ml-notes

> My personal ML / CV learning notes.

Reading notes on papers, concepts I want to think through,
mistakes I made and figured out. Written for future-me, but public
in case anyone finds it useful.

Currently focused on CV — detection and open-vocabulary stuff —
but anything ML-related might show up here depending on what I'm
thinking about.

## Layout

All notes live under [`notes/`](./notes/). Filename prefix tells you the kind:

- `paper-*.md` — paper reading notes
- `concept-*.md` — working through a concept until it clicks
- `note-*.md` — miscellaneous

## On the content

These are learning notes, so expect things that are wrong or
half-understood. If you spot something, feel free to open an issue.

Opinions here are my own and don't represent any organization.
